Unified Modeling Language (UML)

Mis on UML?

UML (Unified Modeling Language) ehk ühtne modelleerimiskeel on standardne visuaalne keel tarkvarasüsteemide modelleerimiseks, kirjeldamiseks ja dokumenteerimiseks.

UML võimaldab arendajatel, analüütikutel ja klientidel paremini mõista süsteemi struktuuri ning käitumist enne programmeerimist.

UML diagrammide loetelu

  • ERD Diagram
  • Flowchart / Activity Diagram
  • Use Case Diagram
  • Class Diagram
  • Sequence Diagram
  • State Diagram
  • Component Diagram

Use Case Diagram

Kirjeldab süsteemi kasutajaid (aktoreid) ja nende tegevusi süsteemis.

Olulised elemendid

  • Actor – süsteemi kasutaja
  • Use Case – tegevus või funktsioon
  • Association – ühendus kasutaja ja tegevuse vahel

Class Diagram

Näitab klasse, nende atribuute, meetodeid ja seoseid.

Olulised elemendid

  • Class
  • Attribute
  • Method
  • Inheritance
  • Association

Sequence Diagram

Näitab objektide vahelist suhtlust ajateljel.

Olulised elemendid

  • Lifeline
  • Message
  • Activation

State Diagram

Kirjeldab objekti erinevaid olekuid ja üleminekuid nende vahel.

Olulised elemendid

  • Initial State
  • State
  • Transition
  • Final State

Component Diagram

Näitab süsteemi komponente ja nendevahelisi sõltuvusi.

Olulised elemendid

  • Component
  • Interface
  • Dependency
